在本章中, 我们将设计一款极其简单的编程语言: `first-step`, 并使用 C++ 实现一个可将 `first-step` 编译到 RISC-V 汇编的编译器.

我们会尽量使用简短而清晰的代码描述整个编译器的结构, 编译器实现的总代码量不超过 300 行.

希望这个例子可以让你对 “如何设计编译器” 这件事情有一个具体的认识.
